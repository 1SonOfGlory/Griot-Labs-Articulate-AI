# Griot Labs - Articlutae AI Brand Starter

## 1) Brand Position
- Company: Griot Labs
- Product: Articlutae AI
- Category: Voice-first communication coaching
- Promise: Turn raw speech into clear, confident, high-impact communication.

## 2) Brand Personality
- Precise
- Modern
- Calm under pressure
- Premium but approachable

## 3) Voice and Tone
- Direct, intelligent, and constructive
- Avoid hype language and empty superlatives
- Use concise statements with clear outcomes
- Favor confidence over excitement

## 4) Color System (v1)
Primary palette:
- `Midnight`: `#061321` (base background)
- `Deep Ocean`: `#0D1F34` (secondary background)
- `Signal Mint`: `#52F6C6` (primary accent)
- `Electric Blue`: `#34A6FF` (secondary accent)
- `Frost`: `#ECF4FF` (main text)
- `Slate Mist`: `#A7BBD4` (supporting text)

UI tokens:
- Glass Surface: `rgba(255, 255, 255, 0.11)`
- Border/Stroke: `rgba(255, 255, 255, 0.22)`
- Shadow: `0 18px 55px rgba(0, 0, 0, 0.32)`

## 5) Typography
- Display / Headlines: `Space Grotesk`
- Body / UI Copy: `Manrope`
- Tone rule: short headline lines, medium-length supporting copy.

## 6) Visual Language
- Glassmorphism cards with soft blur
- Dynamic ambient gradients (subtle movement)
- Rounded controls (friendly, premium)
- Motion should support focus, never distract

## 7) Components (MVP)
- Primary button: Mint -> Blue gradient
- Secondary button: transparent with frosted border
- Hero metrics cards: compact and data-forward
- Section cards: short copy, one insight per card

## 8) Accessibility Guardrails
- Minimum normal text contrast target: WCAG AA where possible
- Keep body text >= 16px equivalent
- Ensure keyboard focus states are visible
- Never rely on color alone for meaning

## 9) Copy Style Guide
- Prefer: "Improve message clarity in high-stakes moments."
- Avoid: "Revolutionize your communication forever!"
- Prefer specific outcomes and context (pitch, negotiation, keynote)

## 10) Next Evolution (when ready)
- Add logo system (wordmark + symbol)
- Add expanded neutrals and semantic colors (success/warning/error)
- Add iconography style and spacing scale
- Add full component library and page templates
